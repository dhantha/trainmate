09 01 04 B2

Service Communication Control Unit
 
Applicable Cars
C Car, S Car, T Car
Equipment Condition
Wheels chocked.
No cab active.
Communication Control Unit powered and active.
Reference Documents
Replace Communication Control Unit – 09 01 04 H
Special Tools
New Jersey Transit Coach Key
Install Communication Control Unit Software Package
 
Note
If performing the activities of this procedure determines that corrective maintenance is required, report it to supervision to ensure that proper corrective action is taken.
 
 
Note
Communication Control Unit (CCU) software service procedure is typical and also applies to destination signs.
 
 
Note
When performing maintenance, finding access doors unlocked on normally-secured lockers or cabinets must be reported to supervisor.
 
  1. Unlock and open locker L1 access door using NJ Transit coach key.
  1. Remove electronics rack shield/cover
Warning
Cables connected to locker equipment can cause electrical shock. Exercise care whenever accessing any of the electrical lockers. Failure to do so may result in injury or death.
 
Warning
The following steps are performed while some locker equipment may be powered. Exercise care whenever near powered equipment. Failure to do so may result in injury or death.
 
Caution
CCU software may be updated using PCMCIA card, PTE, or MDU. PCMCIA card is the fastest way to update software, followed by the PTE, and finally by the MDU. Update CCU software using only one method.
 
 
Note
Updating software using MDU may be very long for large data and is practically only used for very small software package.
 
Update CCU Software Using PCMCIA Card
  1. Ensure that no files CCU_DYZ.tar.gz or XYZ.lzw are in root directory of PCMCIA card. If present, move files to another folder or delete.
  1. Copy software update package onto root directory of compatible PCMCIA card.
  1. Insert PCMCIA card into PCMCIA slot on CCU front panel.
 
Note
PCMCIA LED blinks quickly to indicate that CCU is looking for Software Configuration Item (SCI) update package. When LED blinks slowly, it means that CCU is beginning to update software package.
 
  1. Once PCMCIA LED stops blinking, remove PCMCIA card and proceed to activity “Verify Software Update and Close Locker”.
Update CCU Software Using PTE
  1. Connect PTE COM port to CCU Port using serial cable.


CCU PTE - Main Page


  1. Using PTE Windows, reach Network and Dial-up Connections and double-click PTE connection to CCU (possibly named CCU PTE) to reach PTE CCU main page.


CCU PTE - SCI Update


  1. In CCU PTE main page left menu, click on SCI Update to reach SCI Update screen.
  1. To upload SCI package onto CCU, click link “> Click Here” to open standard browser window.


CCU PTE - SCI Update, Upload in CCU


  1. Use mouse to drag the SCI update package in ftp folder window.
  1. Wait until all softwares in SCI software package are uploaded and transfer completed.
 
Note
Depending on which software is installed, it can take several minutes for update and CCU may reboot by itself once updated.
 
  1. Click Start SCI Update button at bottom of SCI Update screen to start update.
Update CCU Software Using MCU
  1. Put software update in following directory of PCMCIA Card: \ml\swpkg\
  1. Insert PCMCIA card with access to Engineering Area, in MCU slot.


MDU Train Overview Screen


  1. From MDU Train Overview screen, press softkey Eng. Area.


Engineering Area Screen


  1. From MDU Engineering Area screen, press softkey Software Update.


MDU Software Update Screen


  1. From MDU Software Update - Software package Selection screen, use MDU navigation keys to select car system CCU PCMCIA software packages, click PCMCIA softkey button in Source Media group, then click MDU key E to accept.


MDU Software - SCIs To Update


  1. From MDU Software Update - SCIs To Update screen, click Select All softkey button under Select Car System group, then click Start Update softkey button.


MDU Software Update Status Screen


  1. Watch software update progress from MDU Software Update - Status screen, clicking Confirm Install softkey button as required.
Verify Software Update and Close Locker
  1. Reboot CCU following SCI Package update.


CCU PTE - SCI Update, SCI Version


  1. Verify SCI Version numbers with PTE to ensure that the updates are successful using "SCI Versions" in left menu.


CCU PTE - Event Log (figure 38 page 44)


  1. Verify log with PTE using "Get Event Log" in left menu to ensure that no errors occurred during software upload.
  1. Remove PCMCIA Card Key if used.
  1. Replace electronics rack shield/cover.
  1. Close and lock locker L1 access door using NJ Transit coach key.
End of Service
